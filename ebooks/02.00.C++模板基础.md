# 2. C++模板基础

记得2011年左右，我录制了《C++模板视频教程》，专门讲解C++模板技术。（这不是广告，这不是广告，这不是广告！）


要分析Boost代码，光靠基础的C++知识是不够的，还得对模板有一定了解。实际这一章讲解的是纯C++技术，如果你对C++很熟悉，
你完全可以略过这一章。

如果你看下面的代码毫无压力，请继续往下看：

    template<class T>
    T max(T a1, T a2)
    {
        return a1>a2 ? a1 : a2;
    }


我假设你已经了解了模板是什么，因为我们后面将要讲到
- 特化/偏特化
- 萃取
- 等

只有了解了这些，各位看官才能在Boost的世界里，驰骋翱翔。
