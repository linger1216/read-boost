
# 2. C++模板基础

记得2011年左右，我录制了《C++模板视频教程》，专门讲解C++模板技术。（这不是广告，这不是广告，这不是广告！）


要分析Boost代码，光靠基础的C++知识是不够的，还得对模板有一定了解。这一章，实际上是纯C++技术，如果你对C++很熟悉
那么完全可以略过这一章，只有了解这些模板知识，你才能Boost的世界里，驰骋翱翔。


如果你看下面的代码毫无压力，请继续往下看：

    template<class T>
    T max(T a1, T a2)
    {
        return a1>a2 ? a1 : a2;
